What is playfair?

  • Playfair cipher also known as Playfair Square.
  • It is a Cryptographic technique used for manual encryption of data.
  • Playfair Cipher was invented by Charles Wheatstone in 1854.
  • This technique is quite fast and used to protect important information.

How Playfair Cipher is used?

  • Playfair Cipher was used by the British army during World War 1.,
  • Playfair algorithm is based on the use of a (5x5) matrix of letters constructed using a KEYWORD.
  • The letter [I and JJ is count as one letter.

Example

Keyword = PLAYFAIR EXAMPLE

Plaintext = my name is pixy

Rules:

  1. Enter the keyword in the (5x5) matrix row-wise [left-to-right] & then [top-to-bottom].
  2. Drop Duplicate Letters.
  3. Fill remaining space with rest of the letters [A-Z] that were not part of KEYWORD.

Encryption Process

Plaintext: my name is pixy

  • Break down the Plaintext into pairs of 2 Alphabets. Plaintext Becomes: my na me is pi xy

  • If both alphabets are the same or only one is left, add X after the first Alphabet. Only one means: suppose plaintext (see - se ex)
  • If both alphabets appear in the same row of our matrix, replace them with alphabets next to their immediate right respectively.
  • If both alphabets appear in the same column, replace them immediately below them respectively.
  • If alphabets are not in the same row/column, replace them with the text diagonally opposite to it in the row.

  • The ciphertext is xf ol ix km ib gx

Decryption Process

- Opposite steps that we used in Encryption

Cipher text = XF OL IX MK IB GX

  • If both alphabets Appear in same Row of our matrix, Replace them with Alphabets next to their immediate LEFT respectively.
  • If both Alphabets Appear in same Columns, replace them immediately ABOVE them respectively.
  • If alphabets is not in same Row / Column then replace them with the text diagonally opposite to it in the row.




  • The Plaintext is my na me is pi xy
